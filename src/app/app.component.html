Introduction

This app is a preview of the default SQ-UI theme. The UI-kit comes with it but including it into your own app is optional. In case you would like to use this theme, you can do one of the following things:

  • put the sq-ui-theme.scss file in the "styles" array in your angular.json file
  • reference the sq-ui-theme.scss file in your main stylesheet (if you're using sass)

After including the stylesheet, you can use the theme by adding "sq" as a class to the container of your app.

The font used in this preview is "Quicksand" and is applied using the Google Fonts API. The SQ-UI theme does not come with it and does not have a default font set.
